Weather webscrape

A small program which webscrapes weather.com and stores it in a variable. After storing the temprature of the place in this case Turkey Istanbul Park and then another variable stores in the data of the summary of the weather report.

Modlues used

Many modules are required for this project which are not installed in python by default. These modules are as follows:

  1. Starting gitignore

    2. Beautiful soup 4
    Installaion code: pip install beautifulsoup4

  2. Requests

    3. Requests
    Installaion code: pip install requests

  3. lxml

    4. lxml
    Installaion code: pip install lxml
    Note: lxml is not imported into the code but still is essential because the current format doesn't allow processing without lxml imported into the code.

  4. HTML5LIB

    5. html5lib Installaion code: pip install html5lib

  5. OS module

    6. OS Installaion code: Pre-installed, no installation required

  6. Smtplib

    7. Smtplib
    Installaion code: Pre-installed, no installation required
